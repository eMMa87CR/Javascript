Concatenación y Operaciones en JavaScript

Concatenación

La concatenación en JavaScript se utiliza para unir dos o más cadenas de texto. Esto se puede hacer utilizando el operador +.

let nombre = "Juan";
let apellido = "Pérez";
let nombreCompleto = nombre + " " + apellido;
console.log(nombreCompleto); // Juan Pérez

Operaciones

En JavaScript, puedes realizar varias operaciones aritméticas como suma, resta, multiplicación y división.

Suma

let a = 10;
let b = 5;
let suma = a + b;
console.log(suma); // 15

Resta

let c = 20;
let d = 5;
let resta = c - d;
console.log(resta); // 15

Multiplicación

let e = 4;
let f = 5;
let multiplicacion = e * f;
console.log(multiplicacion); // 20

División

let g = 20;
let h = 4;
let division = g / h;
console.log(division); // 5
