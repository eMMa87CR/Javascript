Variables en JavaScript

¿Qué es una variable?

En programación, una variable es un espacio de memoria que se reserva para almacenar un valor. Este valor puede ser modificado durante la ejecución del programa.

Declaración de Variables

En JavaScript, las variables se pueden declarar usando var, let y const.

var

La declaración con var tiene un alcance global o de función y puede ser reasignada y redeclarada.

var nombre = "Juan";
console.log(nombre); // Juan

var nombre = "Pedro";
console.log(nombre); // Pedro

let

La declaración con let tiene un alcance de bloque y puede ser reasignada, pero no redeclarada en el mismo ámbito.

let edad = 30;
console.log(edad); // 30

edad = 31;
console.log(edad); // 31

const

La declaración con const también tiene un alcance de bloque, pero no puede ser reasignada ni redeclarada. Sin embargo, los objetos y arrays declarados con const pueden tener sus propiedades modificadas.

const pi = 3.1416;
console.log(pi); // 3.1416

const persona = { nombre: "Ana", edad: 25 };
persona.edad = 26;
console.log(persona); // { nombre: "Ana", edad: 26 }

Tipos de Datos

Las variables en JavaScript pueden almacenar diferentes tipos de datos, como números, cadenas de texto, booleanos, arreglos, objetos, funciones, etc.

var numero = 10;
let cadena = "Hola, mundo!";
const booleano = true;

Mostrar Variables en la Consola

Para mostrar el valor de una variable en la consola del navegador, se utiliza la función console.log().

console.log(numero);
console.log(cadena);
console.log(booleano);
