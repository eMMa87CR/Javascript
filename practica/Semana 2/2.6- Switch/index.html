Bloque Switch

El bloque switch es una estructura de control que permite evaluar una expresión y compararla con diferentes valores. Si la expresión coincide con alguno de los valores, se ejecuta el bloque de código correspondiente.

La estructura de un bloque switch es la siguiente:

            
                switch (expresion) {
                    case valor1:
                        // Bloque de código
                        break;
                    case valor2:
                        // Bloque de código
                        break;
                    case valor3:
                        // Bloque de código
                        break;
                    default:
                        // Bloque de código
                        break;
                }

La expresión es evaluada una sola vez y su valor es comparado con los valores de los casos. Si la expresión coincide con un valor, se ejecuta el bloque de código correspondiente. Si no coincide con ninguno de los valores, se ejecuta el bloque de código del caso default.
