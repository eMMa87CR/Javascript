Bucle While

Ciclo While

El ciclo while se ejecuta mientras la condición sea verdadera.

Sintaxis: 

            while (condición) {
                // Código a ejecutar
            }

Ejemplo: 

            let i = 0;
            while (i < 5) {
                console.log(i);
                i++;
            }

Ciclo Do While

El ciclo do while se ejecuta al menos una vez y luego se ejecuta mientras la condición sea verdadera.

Sintaxis: 

            do {
                // Código a ejecutar
            } while (condición);

Ejemplo: 

            let numero = 0;
            do {
                numero = (window.prompt("Ingrese un número"));
                console.log("do while", numero);
            } while (parseInt(numero));

While no se ejecuta

En el siguiente ejemplo, la condición del ciclo while no se cumple, por lo que el ciclo no se ejecuta. 

        let j = 7;
        while (j<6){
            console.log("while no se ejecuta",j);
            j++;
        }
