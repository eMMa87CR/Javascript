  • ¿Qué tiene que hacer este código?
  • ¿Por qué no cumple con su función?

Respuesta

El código tiene como objetivo solicitar al usuario que ingrese una cantidad de repeticiones y un texto a repetir. Luego, intenta repetir el texto ingresado la cantidad de veces especificada por el usuario.

El código no cumple con su función debido a dos razones principales:

  • 1. Conversión de Tipo: La función `prompt` devuelve una cadena de texto (`string`), pero el bucle `for` espera que `cantidad` sea un número (`number`) para poder realizar la comparación y la iteración correctamente. Es necesario convertir `cantidad` a un número usando `parseInt` o `Number`.
  • 2. Incremento del Índice: En el bucle `for`, el índice `index` nunca se incrementa dentro del bucle, lo que resulta en un bucle infinito si el código se ejecutara correctamente. Se necesita añadir `index++` o `index += 1` para incrementar el valor de `index` en cada iteración.

Aquí está el código corregido: 
        let cantidad = parseInt(prompt('INGRESE CANTIDAD DE REPETICIONES'), 10);
        let texto = prompt('INGRESE TEXTO A REPETIR');
        for (let index = 0; index < cantidad; index++) {
            console.log(texto);
        }
